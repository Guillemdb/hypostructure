# KRNL-StiffPairing v2: Non-Degeneracy in Constraint Systems

## Overview

This document provides an alternative complexity-theoretic translation of the KRNL-StiffPairing theorem, focusing on **non-degeneracy** as the central organizing principle. While the primary translation (v1) emphasizes matrix rigidity and tensor rank, this perspective highlights connections to constraint satisfaction, linear programming duality, and feasibility theory.

**Core Insight:** Under dissipation conditions, no null directions exist in tangent space; the bilinear form is non-degenerate. In complexity terms: constraint systems have full rank.

**Original Theorem Reference:** {prf:ref}`mt-krnl-stiff-pairing`

---

## Original Statement (Hypostructure)

Let $\mathbb{H} = (\mathcal{X}, \Phi, \mathfrak{D})$ be a hypostructure with bilinear pairing $\langle \cdot, \cdot \rangle : \mathcal{X} \times \mathcal{X} \to F$ such that $\Phi$ is generated by this pairing. Given a decomposition $\mathcal{X} = X_{\mathrm{free}} \oplus X_{\mathrm{obs}} \oplus X_{\mathrm{rest}}$ into free sector, obstruction sector, and possible null sector, if:

1. The pairing is non-degenerate on $X_{\mathrm{free}} \oplus X_{\mathrm{obs}}$
2. Flat directions for $\Phi$ are flat directions for the pairing
3. Any vector orthogonal to $X_{\mathrm{free}}$ lies in $X_{\mathrm{obs}}$

Then $X_{\mathrm{rest}} = 0$: there are no hidden "null modes" and all degrees of freedom are accounted for by free components plus obstructions.

---

## Complexity Theory Statement (Non-Degeneracy Form)

**Theorem (Full Rank Constraint Satisfaction).** Let $\mathcal{C} = (A, b, c)$ be a constraint system over a field $F$, where $A \in F^{m \times n}$ is the constraint matrix, $b \in F^m$ the right-hand side, and $c \in F^n$ a cost vector. Define:

- **Primal feasible region:** $\mathcal{P} = \{x \in F^n : Ax = b, x \geq 0\}$
- **Dual feasible region:** $\mathcal{D} = \{y \in F^m : A^T y \leq c\}$

The constraint system $\mathcal{C}$ is **non-degenerate** if:

1. $\mathrm{rank}(A) = m$ (full row rank: all constraints are independent)
2. Every basic feasible solution has exactly $m$ active constraints
3. No basic feasible solution has a zero-valued basic variable

**Conclusion (Non-Degeneracy Trichotomy).** Under non-degeneracy:

**(1)** The system exhibits a clean trichotomy: exactly one of (a) unique optimal solution, (b) unbounded, or (c) infeasible holds.

**(2)** No degenerate pivots occur in simplex method execution.

**(3)** Strong duality holds with equality: if primal and dual are both feasible, then $c^T x^* = b^T y^*$.

## LP Feasibility and Duality Connections

### Strong Duality as Non-Degeneracy

**Theorem (LP Duality).** For the primal-dual pair:

$$\text{(P)} \quad \min c^T x \text{ s.t. } Ax = b, x \geq 0$$
$$\text{(D)} \quad \max b^T y \text{ s.t. } A^T y \leq c$$

Strong duality holds: if both (P) and (D) are feasible, then $\min_P = \max_D$.

**Non-Degeneracy Correspondence:**

| Stiff Pairing Property | Duality Property |
|------------------------|------------------|
| $\langle x, y \rangle = 0 \Rightarrow x = 0$ or $y = 0$ | No duality gap: $c^T x^* = b^T y^*$ |
| Trivial radical: $\mathrm{rad}(\langle \cdot, \cdot \rangle) = 0$ | Unique optimal face (when bounded) |
| $X_{\mathrm{rest}} = 0$ | No redundant constraints |
| Flat = flat in pairing | Complementary slackness characterization |

### Farkas Lemma as Obstruction Theory

**Lemma (Farkas).** Exactly one of the following holds:
1. $\exists x \geq 0 : Ax = b$ (primal feasible)
2. $\exists y : A^T y \geq 0, b^T y < 0$ (infeasibility certificate)

**Hypostructure Interpretation:** The Farkas alternative corresponds to the obstruction sector $X_{\mathrm{obs}}$:

- Primal feasibility: the obstruction sector accounts for all constraints
- Farkas certificate: an element $y$ in the dual obstruction sector witnesses infeasibility
- Non-degeneracy ensures this dichotomy is clean (no null modes hide additional structure)

## Proof Sketch: Constraint Satisfaction Perspective

### Setup: Linear Constraint Systems

Let $\mathcal{C} = (A, b, c)$ be a linear program with:

- **Constraint matrix** $A \in \mathbb{R}^{m \times n}$ (or over any field $F$)
- **Right-hand side** $b \in \mathbb{R}^m$
- **Objective coefficients** $c \in \mathbb{R}^n$

Define the **constraint form**:
$$Q_A(x) = \|Ax - b\|^2 = x^T A^T A x - 2 b^T A x + \|b\|^2$$

The Hessian $H = 2A^T A$ captures the "stiffness" of constraints.

### Step 1: Full Rank as Non-Degeneracy

**Definition (Non-Degenerate LP).** An LP is non-degenerate if:

1. $\mathrm{rank}(A) = m$ (all constraint rows are independent)
2. For every basic feasible solution, all basic variables are strictly positive
3. No constraint is redundant

**Correspondence to Stiff Pairing:**

The constraint bilinear form $B(x, y) = x^T A^T A y$ satisfies:

$$\ker(B) = \ker(A^T A) = \ker(A)$$

Non-degeneracy of $B$ on the constrained subspace means:

$$\ker(A) \cap \mathrm{col}(A^T) = \{0\}$$

This is exactly the condition that $A$ has full row rank.

### Step 2: Variable Decomposition as Sector Decomposition

**Basic/Non-Basic Partition.** Given a basis $B \subseteq \{1, \ldots, n\}$ with $|B| = m$:

$$x = x_B + x_N$$

where $x_B$ are basic variables and $x_N$ are non-basic.

**Sector Correspondence:**

| LP Concept | Hypostructure Sector |
|------------|---------------------|
| Basic variables $x_B$ | Obstruction sector $X_{\mathrm{obs}}$ |
| Non-basic variables $x_N$ | Free sector $X_{\mathrm{free}}$ |
| Slack in degenerate BFS | Null sector $X_{\mathrm{rest}}$ |

**Non-Degeneracy Condition:** A basic feasible solution is non-degenerate if and only if $X_{\mathrm{rest}} = 0$, i.e., no basic variable equals zero.

### Step 3: Simplex Method and Dissipation

**Simplex Dynamics.** The simplex method defines a "flow" on the space of basic feasible solutions:

$$\text{BFS}_k \xrightarrow{\text{pivot}} \text{BFS}_{k+1}$$

**Dissipation Property.** The objective value strictly decreases (for minimization) with each pivot:

$$c^T x_{k+1} < c^T x_k$$

unless the current BFS is optimal or a degenerate pivot occurs.

**Non-Degeneracy Guarantees:**

1. **No cycling:** The simplex method cannot return to a previously visited BFS
2. **Finite termination:** At most $\binom{n}{m}$ pivots before reaching optimum
3. **Strict descent:** Every pivot strictly improves the objective

**Correspondence to Hypostructure:**

| Simplex Property | Hypostructure Property |
|------------------|------------------------|
| Objective decrease | Energy dissipation $\Phi(S_t x) < \Phi(x)$ |
| No cycling | No periodic orbits outside fixed points |
| Finite termination | Finite convergence to equilibrium |
| Reduced cost = 0 at optimum | Gradient = 0 at fixed point |

### Step 4: Complementary Slackness as Gradient Consistency

**Theorem (Complementary Slackness).** $(x^*, y^*)$ are optimal primal-dual solutions if and only if:

$$x_j^* > 0 \Rightarrow (A^T y^*)_j = c_j$$
$$(A^T y^*)_j < c_j \Rightarrow x_j^* = 0$$

**Gradient Consistency Correspondence:** This is the constraint satisfaction analog of $K_{\mathrm{GC}_\nabla}^+$:

- Flat directions for $\Phi$ (objective) correspond to zero reduced cost: $\bar{c}_j = c_j - (A^T y)_j = 0$
- Flat directions for the pairing correspond to non-binding constraints
- Consistency means: a variable is positive only if its constraint is tight

### Step 5: Radical Analysis and Redundancy Detection

**Definition (Constraint Radical).** Define:

$$\mathrm{rad}(A, b) = \{(\alpha, \beta) \in \mathbb{R}^m \times \mathbb{R} : \alpha^T A = 0, \alpha^T b = 0\}$$

This captures redundant constraint combinations.

**Non-Trivial Radical = Redundant Constraints:**

If $\mathrm{rad}(A, b) \neq \{0\}$, there exists a non-trivial linear combination of constraints that yields $0 = 0$. This indicates:

1. **Primal:** Some constraint is implied by others
2. **Dual:** Dual variables are not uniquely determined
3. **Sector:** $X_{\mathrm{rest}} \neq 0$ (hidden null space)

**Stiff Pairing Guarantees:** When $\mathrm{rad}(A, b) = 0$:

1. All constraints are independent
2. Dual solution is unique (if it exists)
3. The LP decomposition $n = m + (n - m)$ is exact (basic + non-basic = total)

## Certificate Construction

### LP Optimality Certificate

$$K_{\mathrm{LP}}^+ = (x^*, y^*, s^*, \pi_{\mathrm{feas}}, \pi_{\mathrm{comp}})$$

where:
- $x^*$: Primal optimal solution
- $y^*$: Dual optimal solution
- $s^* = c - A^T y^*$: Dual slack vector
- $\pi_{\mathrm{feas}}$: Proof of primal and dual feasibility
- $\pi_{\mathrm{comp}}$: Proof of complementary slackness: $x^* \odot s^* = 0$

**Verification (Polynomial Time):**

1. Check $Ax^* = b$ and $x^* \geq 0$ (primal feasibility)
2. Check $A^T y^* + s^* = c$ and $s^* \geq 0$ (dual feasibility)
3. Check $x_j^* s_j^* = 0$ for all $j$ (complementary slackness)
4. Conclude $c^T x^* = b^T y^*$ (strong duality, optimality)

**Complexity:** $O(mn)$ arithmetic operations.

### Non-Degeneracy Certificate

$$K_{\mathrm{NonDeg}}^+ = (B, x_B^*, \pi_{\mathrm{basis}}, \pi_{\mathrm{strict}})$$

where:
- $B \subseteq \{1, \ldots, n\}$: Optimal basis with $|B| = m$
- $x_B^*$: Basic variable values
- $\pi_{\mathrm{basis}}$: Proof that $A_B$ is invertible (determinant or inverse)
- $\pi_{\mathrm{strict}}$: Proof that $x_B^* > 0$ componentwise

**Verification:**

1. Check $|B| = m$ and $\det(A_B) \neq 0$
2. Verify $x_B^* = A_B^{-1} b > 0$
3. Confirm non-basic variables: $x_N^* = 0$

## Connections to Classical Results

### 1. Linear Independence and Gaussian Elimination

**Theorem.** A matrix $A \in F^{m \times n}$ has full row rank if and only if Gaussian elimination produces $m$ pivot positions without zero rows.

**Connection to Stiff Pairing:** The row echelon form of $A$ reveals:

| Echelon Property | Sector Correspondence |
|------------------|----------------------|
| Pivot columns | Obstruction sector $X_{\mathrm{obs}}$ |
| Free columns | Free sector $X_{\mathrm{free}}$ |
| Zero rows | Null sector $X_{\mathrm{rest}}$ (should be empty) |

Full row rank means no zero rows, i.e., $X_{\mathrm{rest}} = 0$.

### 2. Non-Singular Matrices and Cramer's Rule

**Theorem (Cramer).** For non-singular $A \in F^{n \times n}$ and $b \in F^n$, the unique solution to $Ax = b$ is:

$$x_i = \frac{\det(A_i)}{\det(A)}$$

where $A_i$ is $A$ with column $i$ replaced by $b$.

**Non-Degeneracy Interpretation:** Non-singularity ($\det(A) \neq 0$) is the square-matrix case of non-degeneracy. It guarantees:

1. **Existence:** A solution exists
2. **Uniqueness:** The solution is unique
3. **Computability:** The solution is computable via determinants

### 3. Constraint Qualification in Nonlinear Programming

**Definition (LICQ).** The Linear Independence Constraint Qualification holds at $x^*$ if the gradients of active constraints $\{\nabla g_i(x^*) : g_i(x^*) = 0\}$ are linearly independent.

**Connection to Stiff Pairing:** LICQ is the nonlinear generalization of full-rank constraints:

| LP Non-Degeneracy | LICQ |
|-------------------|------|
| Rows of $A$ at active constraints | Gradients at active constraints |
| Full row rank | Linear independence |
| Unique dual solution | Unique Lagrange multipliers |

### 4. Basis Theorems in Matroid Theory

**Theorem (Matroid Basis).** In a matroid $M = (E, \mathcal{I})$, all bases have the same cardinality.

**Connection:** The columns of a full-rank matrix $A$ form a matroid where:
- Independent sets = linearly independent column subsets
- Bases = maximal independent sets of size $m$

Non-degeneracy in LP corresponds to the matroid property that every basis determines a unique BFS.

### 5. Hodge Theory and Intersection Forms

**Theorem (Hodge Decomposition).** On a compact Kahler manifold:

$$H^k(X, \mathbb{C}) = \bigoplus_{p+q=k} H^{p,q}(X)$$

The intersection pairing on middle cohomology is non-degenerate (Poincare duality).

**Connection:** The stiff pairing condition in hypostructure generalizes Hodge-theoretic non-degeneracy. In the LP setting:

| Hodge Theory | LP Duality |
|--------------|------------|
| Intersection form | Primal-dual pairing $\langle x, y \rangle = c^T x = b^T y$ |
| Non-degeneracy | Strong duality gap = 0 |
| Hodge decomposition | Primal-dual decomposition |

## Algorithmic Implications

### Degeneracy Detection Algorithm

**Input:** Constraint matrix $A \in F^{m \times n}$, RHS $b \in F^m$

**Output:** Non-degeneracy certificate or degeneracy witness

```
1. Compute rank(A) via Gaussian elimination
2. If rank(A) < m:
     Return DEGENERATE with linearly dependent rows
3. Solve LP to find BFS x*
4. Count positive components: k = |{i : x*_i > 0}|
5. If k < m:
     Return DEGENERATE with zero basic variable
6. Return NON-DEGENERATE with certificate K_NonDeg
```

**Complexity:** $O(\max(m^2 n, \text{LP solve time}))$

### Simplex with Anti-Cycling (Bland's Rule)

Even in degenerate cases, non-degeneracy of the underlying constraint structure (full row rank) enables anti-cycling rules:

**Bland's Rule:** When choosing entering/leaving variables, always select the one with smallest index among ties.

**Guarantee:** No cycling occurs, and simplex terminates finitely.

**Connection to Stiff Pairing:** While individual BFS may be degenerate, the global constraint structure having $X_{\mathrm{rest}} = 0$ (full rank) ensures the feasible region has the correct dimension and topology.

## Summary

The KRNL-StiffPairing theorem, viewed through the lens of constraint satisfaction and LP duality, establishes:

1. **Non-degeneracy = Full Rank:** The absence of null directions in the hypostructure pairing corresponds to the constraint matrix having full row rank.

2. **Sector Decomposition = Variable Classification:** The free/obstruction/null decomposition maps to basic/non-basic/redundant variable classification in LP.

3. **Gradient Consistency = Complementary Slackness:** The alignment of flat directions for energy and pairing corresponds to the KKT conditions.

4. **Strong Duality = No Null Sector:** When $X_{\mathrm{rest}} = 0$, the primal-dual gap vanishes and optimality certificates exist.

5. **Dissipation = Objective Improvement:** The simplex method's strict descent mirrors energy dissipation in hypostructure dynamics.

This perspective reveals that non-degeneracy in LP is not merely a technical convenience but a fundamental structural property ensuring that constraint systems are "stiff"---all degrees of freedom are genuinely constrained, with no hidden slack or redundancy.
