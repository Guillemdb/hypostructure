# Fractal Gas & Physics Reference

This document establishes the rigorous metatheorems and mathematical objects for the **Fractal Gas** solver and its **Physical Instantiations** (Quantum, Gravity, Thermodynamics). It serves as the canonical reference for these domains, migrating key structures from the foundational `hypostructure.md`.

## The Fractal Gas (Solver)

The Fractal Gas is the computational instantiation of the hypostructure framework that explicitly separates three geometric structures: the domain of observation, the space of algorithmic reasoning, and the emergent manifold.

### Mathematical Objects

:::{prf:definition} State Space (X)
:label: def-state-space-fg
The **State Space** is a metric measure space $(X, d_X, \mu_X)$ representing the domain of the problem (the "Territory").
- **Agents:** $w_i \in X$
- **Base Dynamics:** $\mathcal{F}_t$ (gradient descent, physics engine)
:::

:::{prf:definition} Algorithmic Space (Y)
:label: def-algorithmic-space-fg
The **Algorithmic Space** is a normed vector space $(Y, \|\cdot\|_Y)$ equipped with a **Projection Map** $\pi: X \to Y$ (the "Map").
- **Distance:** $d_{\text{alg}}(i, j) := \| \pi(w_i) - \pi(w_j) \|_Y$
- **Role:** Cognitive workspace for companion selection.
:::

:::{prf:definition} Emergent Manifold (M)
:label: def-emergent-manifold-fg
The **Emergent Manifold** is the Riemannian manifold $(M, g_{\text{eff}})$ defined by the **Inverse Diffusion Tensor** of the swarm: $g_{\text{eff}} = D^{-1}$, where $D_{ij}$ is the covariance of swarm dispersal.
:::

## Fractal Set Foundations (Discrete-to-Continuum)

The Fractal Set is the discrete-to-continuum bridge, representing the trace of the swarm as an emergent spacetime.

### Discrete Objects

:::{prf:definition} Fractal Set ($\mathcal{F}$)
:label: def-fractal-set
**Origin:** `hypostructure.md`, Ch 19 (Fractal Set Foundations), Line 33618.
A **Fractal Set** is a tuple $\mathcal{F} = (V, \text{CST}, \text{IG}, \Phi_V, w, \mathcal{L})$:
- **CST:** Causal Spacetime Tree (Temporal Precedence $\prec$)
- **IG:** Information Graph (Spatial Adjacency)
- **Fitness:** $\Phi_V: V \to \mathbb{R}_{\geq 0}$
- **Labels:** $\mathcal{L}$ (Topological types, Gauge labels)
:::

:::{prf:definition} Minimizing Movement Scheme
:label: def-minimizing-movement
**Origin:** `hypostructure.md`, Ch 19, Line 33838.
The **Minimizing Movement** scheme approximates gradient flow via recursive variational optimization:
$$ x_{n+1}^\tau \in \arg\min_{x} \left\{ \Phi(x) + \frac{d(x, x_n^\tau)^2}{2\tau} \right\} $$
This provides a variational interpretation of the discrete time-stepping.
:::

### Thermodynamics & Stability

:::{prf:definition} Logarithmic Sobolev Inequality (LSI)
:label: def-lsi
**Origin:** `hypostructure.md`, Ch 6, Line 4490 (Definition), Line 12006 (Theorem).
A measure $\mu$ satisfies **LSI** with constant $\kappa > 0$ if for all smooth $f$:
$$ \text{Ent}_\mu(f^2) \leq \frac{2}{\kappa} \int |\nabla f|^2 d\mu $$
This controls the rate of convergence to equilibrium and suppresses "bad" topological sectors.
:::

:::{prf:metatheorem} The Spectral Generator
:label: mt-spectral-generator
**Origin:** `hypostructure.md`, Ch 6 (General Theory), Line 11992.
**Statement.** Positive Hessian of dissipation $\nabla^2 \mathfrak{D} \succ 0$ enforces a spectral gap and LSI. This guarantees exponential convergence to the safe manifold and prevents stiffness breakdown (Mode S.D).
:::

:::{prf:metatheorem} LSI for Particle Systems
:label: mt-lsi-particle-systems
**Origin:** `hypostructure.md`, Ch 6 (General Theory), Line 12006.
**Statement.** For interacting particle systems, strict convexity of the confining potential (or repulsive pairwise interactions) implies LSI. By the **Otto-Villani theorem**, LSI implies Tranport-Entropy inequalities (Talagrand), guaranteeing concentration of measure.
:::

:::{prf:metatheorem} The Fisher-Hessian Isomorphism
:label: mt-fisher-hessian-isomorphism
**Origin:** `hypostructure.md`, Ch 14 (Thermodynamics), Line 27838.
**Statement.** For exponential families, the Fisher Information Metric is exactly the Hessian of the log-partition function (generating potential).
:::

:::{prf:metatheorem} The Scalar Curvature Barrier
:label: mt-scalar-curvature-barrier
**Origin:** `hypostructure.md`, Ch 14 (Thermodynamics), Line 27934.
**Statement.** The Scalar Curvature $R$ of the Hessian metric measures interaction strength:
- $R=0 \iff$ Independence (Gaussian)
