# Proof of RESOLVE-AutoAdmit (Automatic Admissibility)

:::{prf:proof}
:label: proof-mt-resolve-auto-admit

**Theorem Reference:** {prf:ref}`mt-resolve-auto-admit`

## Setup and Notation

Let $\mathcal{H} = (\mathcal{X}, \nabla, \Phi_\bullet, \tau, \partial_\bullet)$ be a Hypostructure of type $T$ satisfying the Automation Guarantee (Definition {prf:ref}`def-automation-guarantee`). We are given:

**Thin Kernel Objects:**
- $\mathcal{X}^{\text{thin}} = (\mathcal{X}, d, \mu)$ where:
  - $\mathcal{X}$ is a complete metric space (the state space)
  - $d: \mathcal{X} \times \mathcal{X} \to \mathbb{R}_{\geq 0}$ is the metric
  - $\mu$ is a Radon measure on $\mathcal{X}$
- $\mathfrak{D}^{\text{thin}} = (R, \beta)$ where:
  - $R: \mathcal{X} \to \mathbb{R}_{\geq 0}$ is the dissipation rate density
  - $\beta \in \mathbb{R}_{>0}$ is the scaling exponent for dissipation

**Singularity Data:**
At a potential surgery time $t^* < \infty$, we have identified:
- A singularity point $x^* \in \mathcal{X}$
- A profile $V \in \mathcal{M}_{\text{prof}}(T)$ obtained from the Profile Classification Trichotomy (MT {prf:ref}`mt-resolve-profile`)
- A singular locus $\Sigma \subseteq \mathcal{X}$ to be determined

**Ambient Dimension:**
Let $n = \dim(\mathcal{X})$ denote the topological dimension of the state space (locally constant for manifolds, covering dimension for general spaces).

**Goal:**
We show that, under the Automation Guarantee and the admissibility data recorded for $T$, the Surgery Admissibility Trichotomy ({prf:ref}`mt-resolve-admissibility`) can be **automatically certified** using only the thin objects together with the built-in verification procedures. In particular, the canonicity, codimension, and capacity checks are performed using the provided profile-library predicates, threshold data, and certified numerical routines, rather than user-written proofs.

## Step 1: Singular Locus Identification

### Step 1.1: Definition of the Singular Set

Using the dissipation rate density $R$ from $\mathfrak{D}^{\text{thin}}$, we define the **pre-singular set** as:

$$\Sigma_0 := \{x \in \mathcal{X} : \limsup_{t \nearrow t^*} R(S_t x) = \infty\}$$

where $S_t: \mathcal{X} \to \mathcal{X}$ is the semiflow generated by the connection $\nabla$.

**Justification:** By the energy-dissipation inequality (Axiom D, {prf:ref}`ax-dissipation`):
$$\frac{d}{dt}\Phi(S_t x) = -R(S_t x)$$

Unbounded dissipation along the trajectory is a **sufficient indicator** of singular behavior in the framework. The admissibility filters below only require $\Sigma$ to contain the true singular locus; false positives are removed by the capacity and codimension checks.

### Step 1.2: Metric Structure Extraction

From $\mathcal{X}^{\text{thin}} = (\mathcal{X}, d, \mu)$, we have the metric $d$ which induces:
1. **Metric topology:** The open sets $\{B_\epsilon(x) : x \in \mathcal{X}, \epsilon > 0\}$
2. **Hausdorff measure:** $\mathcal{H}^s$ for dimension $s \in [0, n]$
3. **Sobolev spaces:** $H^1(\mathcal{X}) = \{u \in L^2(\mathcal{X}, \mu) : \nabla u \in L^2(\mathcal{X}, \mu)\}$

where $\nabla$ is the distributional gradient with respect to the metric measure structure $(d, \mu)$.

### Step 1.3: Regularization of the Singular Set

The set $\Sigma_0$ may not be closed. We define the **effective singular locus** as:

$$\Sigma := \overline{\Sigma_0} \cap \{x : R(x) \geq R_{\text{thresh}}\}$$

where $R_{\text{thresh}}$ is a type-dependent threshold recorded in the admissibility data (chosen consistently with the capacity bound).

**Properties of $\Sigma$:**

**Lemma 1.3.1 (Closedness):** $\Sigma$ is closed in $\mathcal{X}$.

*Proof:* The set $\{x : R(x) \geq R_{\text{thresh}}\}$ is closed by lower semicontinuity of $R$ (consequence of dissipation axiom). The intersection of a closed set with a closure is closed. □

**Lemma 1.3.2 (Energy Localization):** Under the bounded-geometry and coarea assumptions recorded in the admissibility data, there exists $\epsilon_0 > 0$ and a constant $C$ depending on $\Phi(x^*)$ and $R_{\text{thresh}}$ such that:
$$\mu(B_\epsilon(\Sigma)) \leq C \cdot \epsilon^2 \quad \text{for all } \epsilon \in (0, \epsilon_0).$$

*Proof:* By the energy-dissipation inequality:
$$\int_0^{t^*} R(S_t x) \, dt \leq \Phi(x_0)$$

The measure of points with $R \geq R_{\text{thresh}}$ is controlled by:
$$\mu(\{x : R(x) \geq R_{\text{thresh}}\}) \leq \frac{\Phi(x_0)}{R_{\text{thresh}}}$$

The $\epsilon$-neighborhood bound follows from the coarea formula and the recorded metric-measure regularity assumptions. □

## Step 2: Canonicity Check (Algorithmic)

### Step 2.1: Profile Extraction from Thin Objects

By the Automation Guarantee (Definition {prf:ref}`def-automation-guarantee`, Item 1), the profile $V$ is computed automatically via a scaling limit certified by the admissibility data:

$$V = \lim_{\lambda \to 0} \lambda^{-\alpha} \cdot S_{t^* + \lambda^2 t}(x^* + \lambda y)$$

where $\alpha$ is the scaling exponent from $\Phi^{\text{thin}}$.

**Algorithmic Implementation (certified numerics):**
1. **Rescaling sequence:** For $\lambda_k = 2^{-k}$, compute:
   $$V_k := \lambda_k^{-\alpha} \cdot S_{t^* + \lambda_k^2 t}(x^* + \lambda_k y)$$
2. **Convergence test:** Use the metric $d$ from $\mathcal{X}^{\text{thin}}$ to check:
   $$d(V_k, V_\ell) \leq \epsilon_{\text{tol}} \quad \text{for all } k, \ell \geq N(\epsilon_{\text{tol}})$$
3. **Limit extraction:** Extract $V = \lim_{k \to \infty} V_k$ in the weak topology of $\mathcal{X}$ using the convergence certificate provided by the automation guarantee.

### Step 2.2: Canonical Library Membership

The canonical library $\mathcal{L}_T$ is **finite** for good types (Remark {prf:ref}`rem-good-types`) and consists of profiles with:
1. Finite automorphism group: $|\text{Aut}(V)| < \infty$
2. Isolated in moduli space: $V$ is a critical point of the profile energy

**Algorithmic Test:** Given $V$ from Step 2.1 and the finite library $\mathcal{L}_T = \{V_1, \ldots, V_N\}$:

**Algorithm 2.2.1 (Library Lookup):**
```
for i = 1 to N do:
    Compute distance: dist_i := d_prof(V, V_i)
    If dist_i < δ_match(T):
        If IsEquivalent(V, V_i, G^thin):
            Return (CANONICAL, i)
        Else:
            Check for equivalence move (YES~)
            Return (CANONICAL_SIM, i, equiv_move)
    End If
End For
Return (NON_CANONICAL)
```

where:
- $d_{\text{prof}}$ is the metric on the profile moduli space $\mathcal{M}_{\text{prof}}(T)$
- $\delta_{\text{match}}(T)$ is the isolation radius for library profiles
- `IsEquivalent` checks whether $V$ and $V_i$ are in the same $G$-orbit

**Lemma 2.2.2 (Decidability of Canonicity):** For good types $T$ equipped with admissibility data, the canonicity check terminates in finite time.

*Proof:* The library $\mathcal{L}_T$ is finite by definition. Each distance computation $d_{\text{prof}}(V, V_i)$ is computable from the thin objects:
$$d_{\text{prof}}(V, W) = \inf_{g \in G} \|g \cdot V - W\|_{H^{s_c}}$$

where $H^{s_c}$ is the critical Sobolev space determined by $\Phi^{\text{thin}}$. The admissibility data supplies a certified routine for approximating the $G$-orbit distance and deciding membership up to the tolerance $\delta_{\text{match}}(T)$. □

## Step 3: Codimension Bound (Automatic)

### Step 3.1: Hausdorff Dimension Computation

The codimension requirement is:
$$\text{codim}(\Sigma) = n - \dim_H(\Sigma) \geq 2$$

where $\dim_H(\Sigma)$ is the Hausdorff dimension of $\Sigma$.

**Computation Strategy:** We use the capacity-dimension relationship from geometric measure theory.

**Theorem 3.1.1 (Capacity-Dimension Bound, {cite}`Federer69`):** Under the doubling and Poincaré-type hypotheses recorded in the admissibility data, for any compact set $\Sigma \subset \mathcal{X}$ in a metric measure space $(\mathcal{X}, d, \mu)$ of dimension $n$:
$$\text{Cap}_{H^1}(\Sigma) < \infty \quad \Rightarrow \quad \dim_H(\Sigma) \leq n - 2$$

where $\text{Cap}_{H^1}$ is the Sobolev 1-capacity defined by:
$$\text{Cap}_{H^1}(\Sigma) := \inf\left\{\int_\mathcal{X} (|\nabla \phi|^2 + \phi^2) \, d\mu : \phi \in H^1(\mathcal{X}), \phi|_\Sigma \geq 1\right\}$$

**Algorithmic Implementation (certified numerics):**

**Algorithm 3.1.2 (Codimension Verification):**
1. **Capacity computation:** Solve the variational problem:
   $$\phi^* = \arg\min\left\{\int_\mathcal{X} |\nabla \phi|^2 \, d\mu : \phi \in H^1(\mathcal{X}), \phi|_\Sigma = 1\right\}$$

   This is a convex optimization problem solvable via the numerical routine recorded in the admissibility data (e.g., finite elements on $\mathcal{X}$).

2. **Capacity evaluation:**
   $$\text{Cap}(\Sigma) = \int_\mathcal{X} |\nabla \phi^*|^2 \, d\mu$$

3. **Dimension bound:** If $\text{Cap}(\Sigma) < \infty$, then by Theorem 3.1.1:
   $$\dim_H(\Sigma) \leq n - 2 \quad \Rightarrow \quad \text{codim}(\Sigma) \geq 2$$

**Lemma 3.1.3 (Computability):** The capacity $\text{Cap}(\Sigma)$ is computable from the thin objects together with the certified numerical scheme provided by the admissibility data.

*Proof:* The Sobolev space $H^1(\mathcal{X})$ is defined using only the measure $\mu$ and the metric gradient $\nabla$ induced by $d$. The constraint $\phi|_\Sigma = 1$ is enforced via penalty methods or Lagrange multipliers as specified by the admissibility data. The minimization is numerically stable under the recorded regularity assumptions. □

### Step 3.2: Refined Dimension Estimate via Profile Energy

For types where the profile $V$ has additional structure, we can obtain sharper bounds.

**Proposition 3.2.1 (Profile-Based Codimension Estimate):** If the profile $V \in \mathcal{L}_T$ is a self-similar solution with scaling dimension $\alpha$, then the scaling model recorded in the profile data predicts:
$$\text{codim}(\Sigma) \approx \frac{\alpha n}{\alpha + \beta}$$

where $\beta$ is the dissipation scaling exponent from $\mathfrak{D}^{\text{thin}}$.

*Proof:* By the scaling invariance of the profile:
$$V(\lambda y) = \lambda^{-\alpha} V(y)$$

The dissipation rate scales as:
$$R(V(\lambda y)) = \lambda^{-\beta} R(V(y))$$

The measure of the singular set satisfies:
$$\mu(\Sigma_\lambda) = \lambda^{\dim_H(\Sigma)} \mu(\Sigma)$$

where $\Sigma_\lambda = \{y : V(\lambda y) \in \Sigma\}$.

Matching energy and dissipation scaling (heuristically):
$$\alpha \cdot \dim_H(\Sigma) = \beta \cdot (n - \dim_H(\Sigma))$$

Solving the scaling balance for $\dim_H(\Sigma)$ yields the model prediction:
$$\dim_H(\Sigma) \approx \frac{\beta n}{\alpha + \beta} = n - \frac{\alpha n}{\alpha + \beta}.$$

Therefore, under the scaling model:
$$\text{codim}(\Sigma) \approx n - \dim_H(\Sigma) \approx \frac{\alpha n}{\alpha + \beta}$$

For parabolic PDE with $\alpha \geq \beta$, and requiring $\text{codim}(\Sigma) \geq 2$, we need:
$$\frac{\alpha n}{\alpha + \beta} \geq 2 \quad \Rightarrow \quad \alpha \geq \frac{2\beta}{n-2}$$

which holds for typical parabolic systems when $n \geq 3$ and $\alpha \geq \beta$. This is a consistency check; the admissibility decision still uses the capacity bound. □

## Step 4: Capacity Bound (Sobolev Capacity Formula)

### Step 4.1: Sobolev Capacity Definition

The critical component of the admissibility check is verifying:
$$\text{Cap}(\Sigma) \leq \varepsilon_{\text{adm}}(T)$$

where $\varepsilon_{\text{adm}}(T)$ is the type-dependent admissibility threshold.

**Definition 4.1.1 (Sobolev 2-Capacity):** Following {cite}`AdamsHedberg96`, the $H^1$-capacity of $\Sigma$ is:
$$\text{Cap}(\Sigma) := \inf\left\{\int_\mathcal{X} |\nabla \phi|^2 \, d\mu : \phi \in H^1(\mathcal{X}), \phi|_\Sigma = 1, \text{supp}(\phi) \text{ compact}\right\}$$

This is the key quantity computed from $\mathcal{X}^{\text{thin}} = (\mathcal{X}, d, \mu)$ using the certified numerical routine recorded in the admissibility data.

### Step 4.2: Capacity Computation Algorithm

**Algorithm 4.2.1 (Variational Capacity):**

**Input:** Singular set $\Sigma \subset \mathcal{X}$, metric measure space $(\mathcal{X}, d, \mu)$

**Output:** $\text{Cap}(\Sigma)$ or certificate that capacity is infinite

**Procedure:**
1. **Test function construction:** For each $R > 0$, construct the distance function:
   $$d_\Sigma(x) := \inf_{y \in \Sigma} d(x, y)$$

2. **Regularization:** Smooth $d_\Sigma$ via the regularization method specified by the admissibility data (e.g., convolution or Lipschitz approximation):
   $$\phi_R(x) := \max\left\{1 - \frac{d_\Sigma(x)}{R}, 0\right\}$$

3. **Energy evaluation:** Compute:
   $$E_R := \int_\mathcal{X} |\nabla \phi_R|^2 \, d\mu$$

4. **Limiting behavior:** As $R \to 0^+$:
   $$\text{Cap}(\Sigma) = \lim_{R \to 0^+} E_R$$

5. **Threshold test:** If $\liminf_{R \to 0} E_R \leq \varepsilon_{\text{adm}}(T)$, return `ADMISSIBLE`.

**Lemma 4.2.2 (Convergence of Capacity Approximants):** Under the regularity hypotheses recorded in the admissibility data, the sequence $\{E_R\}_{R>0}$ converges to $\text{Cap}(\Sigma)$.

*Proof:* By the direct method of calculus of variations {cite}`EvansGariepy15`, the infimum in the capacity definition is attained by a unique minimizer $\phi^* \in H^1(\mathcal{X})$. The functions $\phi_R$ form a recovery sequence converging to $\phi^*$ in $H^1$ as $R \to 0$. □

### Step 4.3: Volume-Capacity Relationship

For practical computation, we use the classical estimate from {cite}`AdamsHedberg96`:

**Theorem 4.3.1 (Volume-Capacity Estimate):** In dimension $n \geq 3$, for a compact set $\Sigma$ with smooth boundary in a bounded-geometry setting:
$$\text{Cap}(\Sigma) \asymp \text{Vol}(\Sigma)^{(n-2)/n}$$

where $\sim$ means the ratio is bounded above and below by constants depending only on $n$ and the geometry of $\mathcal{X}$.

**Proof Sketch:**
1. **Lower bound:** Use test function $\phi = 1$ on $\Sigma$, extended harmonically:
   $$\Delta \phi = 0 \quad \text{in } \mathcal{X} \setminus \Sigma, \quad \phi|_\Sigma = 1, \quad \phi \to 0 \text{ at } \infty$$

   The Dirichlet energy satisfies:
   $$\int |\nabla \phi|^2 \geq C \cdot \text{Cap}(\Sigma)$$

2. **Upper bound:** By the isoperimetric inequality in $\mathbb{R}^n$:
   $$\text{Vol}(\Sigma) \leq C \cdot \text{Area}(\partial\Sigma)^{n/(n-1)}$$

   Combined with the co-area formula:
   $$\int |\nabla \phi|^2 = \int_0^1 \text{Area}(\{\phi = t\}) \, dt$$

   This yields:
   $$\text{Cap}(\Sigma) \leq C \cdot \text{Vol}(\Sigma)^{(n-2)/n}$$

The detailed proof is in {cite}`AdamsHedberg96`, Theorem 5.1.2. □

**Algorithmic Consequence:** Instead of solving the full variational problem, we can compute:
1. **Volume estimate:** $\text{Vol}(\Sigma) \approx \mu(\Sigma_\epsilon)$ for small $\epsilon > 0$
2. **Capacity estimate:** $\text{Cap}(\Sigma) \lesssim \mu(\Sigma_\epsilon)^{(n-2)/n}$
3. **Admissibility test:** Check if $\mu(\Sigma_\epsilon)^{(n-2)/n} \leq \varepsilon_{\text{adm}}(T)$

This reduces the capacity computation to a measure evaluation under the recorded bounded-geometry assumptions, using $\mu \in \mathcal{X}^{\text{thin}}$.

## Step 5: Integration into Trichotomy (Automated Decision)

### Step 5.1: Trichotomy Algorithm Assembly

Combining Steps 2-4, we obtain the complete automated algorithm using the admissibility data and automation routines:

**Algorithm 5.1.1 (Automated Admissibility Trichotomy):**

**Input:**
- Thin objects $(\mathcal{X}^{\text{thin}}, \mathfrak{D}^{\text{thin}})$
- Singularity data $(t^*, x^*, V)$ from profile classification

**Output:** Certificate $K \in \{K_{\text{adm}}, K_{\text{adm}}^\sim, K_{\text{inadm}}\}$

**Procedure:**

```pseudocode
1. SINGULAR_LOCUS := IdentifySingularSet(D_thin, t*)  // Step 1
2. PROFILE := ExtractProfile(x*, Phi_thin, G_thin)     // Step 2.1

3. // Canonicity Check
4. (CANONICAL_STATUS, LIBRARY_INDEX, EQUIV_DATA) :=
       CanonicalityCheck(PROFILE, L_T)                // Step 2.2

5. If CANONICAL_STATUS = NON_CANONICAL:
6.     If PROFILE in Horizon(T):
7.         Return K_inadm("Horizon mode", PROFILE)
8.     Else:
9.         Return K_inadm("Non-canonical profile", PROFILE)

10. // Codimension Check
11. CAP_VALUE := ComputeCapacity(SINGULAR_LOCUS, X_thin)  // Step 4.2
12. If CAP_VALUE = ∞:
13.     Return K_inadm("Infinite capacity", SINGULAR_LOCUS)

14. DIM_ESTIMATE := DimensionFromCapacity(CAP_VALUE, n)   // Step 3.1
15. CODIM := n - DIM_ESTIMATE
16. If CODIM < 2:
17.     Return K_inadm("Codimension too small", CODIM)

18. // Capacity Bound Check
19. If CAP_VALUE > ε_adm(T):
20.     Return K_inadm("Capacity exceeds threshold", CAP_VALUE)

21. // All checks passed
22. If CANONICAL_STATUS = CANONICAL:
23.     Return K_adm(LIBRARY_INDEX, EQUIV_DATA, CAP_VALUE)
24. Else:  // CANONICAL_STATUS = CANONICAL_SIM
25.     Return K_adm^~(LIBRARY_INDEX, EQUIV_DATA, CAP_VALUE)
```

### Step 5.2: Correctness and Completeness

**Theorem 5.2.1 (Correctness):** Algorithm 5.1.1 produces a valid admissibility certificate.

*Proof:* Each branch of the algorithm produces a certificate satisfying the schema from {prf:ref}`mt-resolve-admissibility`:

**Case 1 (Lines 22-23):** All checks pass, $V \in \mathcal{L}_T$. The certificate $K_{\text{adm}}$ includes:
- Canonicity witness: $(V, \text{LIBRARY\_INDEX})$
- Codimension bound: $\text{CODIM} \geq 2$
- Capacity bound: $\text{CAP\_VALUE} \leq \varepsilon_{\text{adm}}(T)$
- Progress density: Guaranteed by library profile energy structure

**Case 2 (Lines 24-25):** Equivalence move required. The certificate $K_{\text{adm}}^\sim$ includes:
- Equivalence data: $(\phi, K_{\text{transport}})$ where $\phi: x \to \tilde{x}$ is the equivalence morphism
- Target admissibility: $K_{\text{adm}}[\tilde{x}]$ from recursive call

**Case 3 (Lines 5-9, 12-13, 16-17, 19-20):** A check fails. The certificate $K_{\text{inadm}}$ includes:
- Failure reason: Specific line that triggered the return
- Witness: Explicit data (profile, capacity value, codimension)

All certificates are constructed from data computed from thin objects together with the admissibility data and automation routines. □

**Theorem 5.2.2 (Completeness):** For good types satisfying the Automation Guarantee and supplied admissibility data, Algorithm 5.1.1 terminates and produces exactly one certificate.

*Proof:*
1. **Termination:** Each subroutine terminates:
   - `IdentifySingularSet`: Closed set computation under the admissibility regularity assumptions
   - `ExtractProfile`: Converges by compactness (Automation Guarantee)
   - `CanonicalityCheck`: Finite library search (Lemma 2.2.2)
   - `ComputeCapacity`: Variational problem with certified convergence (Lemma 4.2.2)

2. **Uniqueness:** The algorithm has mutually exclusive branches:
   - Lines 5-9: Non-canonical path (disjoint from canonical)
   - Lines 12-13: Infinite capacity (disjoint from finite capacity)
   - Lines 16-17: Low codimension (disjoint from codim ≥ 2)
   - Lines 19-20: High capacity (disjoint from capacity ≤ threshold)
   - Lines 22-25: Success paths (disjoint from all failure paths)

3. **Exhaustiveness:** Every input follows exactly one branch by trichotomy. □

## Step 6: Certificate Construction

### Step 6.1: Certificate Schema for Case 1 (Admissible)

When all checks pass (Algorithm 5.1.1, Lines 22-23), we construct:

$$K_{\text{adm}} = (\mathsf{mode}, \mathsf{data}, \mathsf{proof}, K_\epsilon^+)$$

where:

**Mode Identifier:** $\mathsf{mode} := \texttt{SURGERY\_ADMISSIBLE}$

**Surgery Data:**
$$\mathsf{data} := (\Sigma, V, t^*, \epsilon_{\text{excise}}, \mathcal{X}_{\text{cap}})$$
where:
- $\Sigma$ is the singular locus from Step 1
- $V = V_{\mathsf{LIBRARY\_INDEX}} \in \mathcal{L}_T$ is the canonical profile
- $t^*$ is the surgery time
- $\epsilon_{\text{excise}}$ is the excision radius recorded in the admissibility data (typically comparable to $\sqrt{\text{Cap}(\Sigma)}$ for geometric flows)
- $\mathcal{X}_{\text{cap}}$ is the cap geometry attached to $V$ (determined by $V$'s asymptotic structure)

**Admissibility Proof:**
$$\mathsf{proof} := (\mathsf{canonicity\_cert}, \mathsf{codim\_cert}, \mathsf{cap\_cert})$$
where:
- $\mathsf{canonicity\_cert} := (V, \mathsf{LIBRARY\_INDEX}, d_{\text{prof}}(V, V_{\mathsf{LIBRARY\_INDEX}}))$
- $\mathsf{codim\_cert} := (\text{CODIM}, \text{DIM\_ESTIMATE}, \mathsf{Federer\_thm\_ref})$
- $\mathsf{cap\_cert} := (\text{CAP\_VALUE}, \varepsilon_{\text{adm}}(T), \phi^*, \|\nabla \phi^*\|_{L^2})$

**Progress Certificate:** $K_\epsilon^+$ witnesses the discrete progress bound:
$$\Delta\Phi_{\text{surg}} := \Phi(x^-) - \Phi(x') \geq \epsilon_T > 0$$

:::{admonition} Type-Specific Formula
:class: note

The specific energy-capacity relationship depends on type $T$. The abstract framework requires $\Delta\Phi_{\text{surg}} \geq f_T(\text{size}(\Sigma))$ for some type-dependent function $f_T$. The formula below is the instantiation for geometric flows.
:::

For geometric flows (Ricci, MCF), this follows from the energy-capacity relationship:
$$\Delta\Phi_{\text{surg}} \geq c(T) \cdot \text{Cap}(\Sigma)^{n/(n-2)}$$

where $c(T) > 0$ is the type-dependent progress constant.

**Type-Specific Instantiations:**

| Type $T$ | Energy-Capacity Relationship | Reference |
|----------|------------------------------|-----------|
| Ricci flow | $\Delta\Phi \geq c \cdot \text{Cap}^{3/1}$ (dim 3) | {cite}`Perelman03` §4 |
| Mean curvature flow | $\Delta\Phi \geq c \cdot \text{Cap}^{n/(n-2)}$ | {cite}`HuiskenSinestrari09` |
| Harmonic map flow | $\Delta\Phi \geq \epsilon_0$ (bubble quantum) | {cite}`Struwe88` |

**Lemma 6.1.1 (Progress Certificate Derivation):** For profiles $V \in \mathcal{L}_T$ with finite capacity, the progress certificate $K_\epsilon^+$ is constructed using the type-specific bounds recorded in the admissibility data.

*Proof:* By the surgery energy bound (type-specific; see table above), the energy drop satisfies an inequality of the form:
$$\Phi(x') \leq \Phi(x^-) - \int_{\Sigma} |\nabla V|^2 \, d\mu + C_T \epsilon_{\text{excise}}^2,$$
with $C_T$ recorded in the admissibility data.

The excision integral satisfies a type-dependent lower bound. For geometric flows with scaling exponent $(n-2)/n$:
$$\int_{\Sigma} |\nabla V|^2 \, d\mu \geq c_0 \cdot \text{Cap}(\Sigma)^{n/(n-2)}$$

where $c_0 = c_0(V)$ depends only on the canonical profile $V$. Since $V$ is from the finite library $\mathcal{L}_T$, the admissibility data records a uniform lower bound:
$$\epsilon_T := \min_{V \in \mathcal{L}_T} \left(f_T(v_{\min}) - E_{\text{cap}}(V)\right) > 0,$$
with the right-hand side certified as positive.

where $f_T$ is the type-specific energy lower bound function. This is the discrete progress constant for type $T$. □

### Step 6.2: Certificate Schema for Case 2 (Admissible after Equivalence)

When an equivalence move is required (Algorithm 5.1.1, Lines 24-25):

$$K_{\text{adm}}^\sim = (K_{\text{equiv}}, K_{\text{transport}}, K_{\text{adm}}[\tilde{x}])$$

where:

**Equivalence Certificate:** $K_{\text{equiv}}$ contains:
- Source state: $x \in \mathcal{X}$
- Target state: $\tilde{x} = \phi(x)$ where $\phi \in \text{Equiv}(T)$ (Definition {prf:ref}`def-equiv-move`)
- Equivalence morphism: $\phi: \mathcal{X} \to \mathcal{X}$ preserving energy up to the admissibility tolerance: $\Phi(\phi(x)) = \Phi(x) + O(\delta_T)$

**Transport Certificate:** $K_{\text{transport}}$ witnesses that structure is preserved:
- Profile transport: $V(\tilde{x}) = \rho_g(V(x))$ for some $g \in G$ (symmetry group)
- Singular locus transport: $\Sigma(\tilde{x}) = \phi(\Sigma(x))$
- Capacity preservation: $\text{Cap}(\phi(\Sigma)) = \text{Cap}(\Sigma) + O(\delta_T)$

**Recursive Admissibility:** $K_{\text{adm}}[\tilde{x}]$ is obtained by applying Algorithm 5.1.1 to $(\tilde{x}, \phi(\Sigma), \rho_g(V))$.

**Lemma 6.2.1 (Termination of Equivalence Recursion):** The recursion depth is bounded by $|\text{Equiv}(T)|$, the size of the equivalence library recorded in the admissibility data.

*Proof:* Each equivalence move changes the state to a "more canonical" representative in the equivalence class. By the well-foundedness of the equivalence order (guaranteed by the admissibility data through a certified complexity decrease), the recursion terminates after at most $|\text{Equiv}(T)| < \infty$ steps. □

- Otherwise: Terminate sieve run with genuine singularity outcome

### Step 7.1: No User Code Required

**Theorem 7.1.1 (Zero User Burden for Admissibility):** Given thin objects satisfying basic consistency and the admissibility data bundled with the framework, Algorithm 5.1.1 requires no additional user-provided verification code.

*Proof:* We verify each component:
